# 台股投資分析 PDF 報告產生器

自動查詢台股公開資料，產出中文 PDF 投資分析報告。

## 功能特色

- **Web 圖形介面** — 瀏覽器操作，輸入股票立即產出報告 + 即時進度顯示
- **命令列模式** — 直接下參數快速產出
- **輸入股票代號或名稱**，自動辨識（支援 250+ 檔上市櫃股票）
- **綜合評級 A/B/C/D**（整合健康度 50% + 估值 25% + 成長 25% 加權，頂端徽章 + 4 段顏色條）
- **ETF 專屬評級**（費用率 35% + 折溢價 25% + 流動性 20% + 殖利率 20%，ETF 模式自動切換）
- **視覺化進度條**（SSE 即時更新百分比，完成後自動淡出）
- **資料完整性 pills**（每項資料以彩色 pill 顯示可用狀態：股價、營收/EPS/NAV、股利、新聞）
- **股價走勢圖**（3 個月 / 6 個月 / 1 年，含高點低點標註，可切換純股價/含均線/含成交量）
- **每月營收分析**（圖表 + 近 6 期營收概況表 + 趨勢說明）
- **EPS 分析**（近 8 季圖表 + 季度明細 + 年度總和）
- **PE 區間估值**（便宜價 / 合理價 / 昂貴價 + 安全邊際）
- **PEG 分析**（本益比與成長率配適度評估）
- **營收成長評估**（趨勢斜率、連續成長月數、動能增減）
- **健康度評分**（0–100 分，含成長、估值、獲利、動能、穩定五維度長條圖 + 對照表）
- **風險提示**（紅/黃/綠燈訊號）
- **股利分析**（歷年配息紀錄、殖利率趨勢、連續配息年數）
- **股價圖疊加均線**（月線/季線/年線 + 高於/低於判斷）
- **新聞摘要**（多來源聚合：Google News、Bing News、MoneyDJ、鉅亨網、經濟日報 + 產業備援）
- **基本資訊**（含本益比、殖利率、ROE 等財務指標 + ETF 自動辨識）
- **公司簡介**（yfinance 長描述、國家、員工人數）
- **財務指標卡片**（毛利率、營業利益率、淨利率、ROE、ROA、負債比、自由現金流、營收成長、Beta、預估本益比、每股淨值）
- **近期新聞**（preview 直接顯示前 5 則新聞標題、摘要、情緒標籤、來源）
- **漲跌幅顯示**（preview header 與主卡片直接顯示今日漲跌幅百分比）
- **市值顯示**（preview 主卡片顯示市值，依規模顯示百億/億/萬）
- **名詞解釋彈窗**（📖 固定按鈕，彈窗顯示 PE/PEG/NAV/折溢價/費用率/AUM 等 13 項白話解釋）
- **Preview 好壞範圍提示**（各數值下方顯示參考範圍）
- **名詞浮動提示**（ⓘ hover 白話解釋）
- **投資檢查清單**（✅/⚠️/❌ Pass/Fail 檢核表，ETF 版本獨立）
- **操作建議區塊**（根據股價位階與基本面給建議，ETF 版本顯示溢折價/費用率/配息建議）
- **股票比較功能**（加入比較列表，並排顯示關鍵指標，重新整理自動清除）
- **歷史查詢**（最近查過股票存 localStorage，點擊重分析，含清除按鈕）
- **同業比較**（同產業 PE/殖利率/市值/營收成長比較，含 6 欄對照表；ETF 模式改為 NAV/折溢價/費用率比較）
- **風險警示分類**（短期／中期／長期欄位分組顯示）
- **個股行事曆**（除息日、財報公布日、股利發放月份）
- **新聞情緒標示**（偏正向/偏保守/中性）
- **新手引導**（按鈕點擊顯示引導 overlay）
- **報酬預估計算機**（1年/3年持有預期報酬率，ETF 模式自動隱藏）
- **CSV 匯出**（單股 + 比較批次匯出）
- **英文介面切換**（右上角 EN/中 按鈕，全頁面翻譯）
- **PWA 支援**（manifest + service worker，可安裝到手機桌面）
- **自訂背景顏色**（頁尾色塊點擊選色，localStorage 記憶）
- **快取容量上限 + LRU**（檔案快取超過 100 個自動淘汰最舊）
- **Output 啟動自動清空**（每次重開 output/ 目錄清空）
- **Favicon**（icon-192.png / icon-512.png，手機/桌面標籤顯示）
- **ETF 完整支援**（0050/0056/00878 等 — 專屬 preview 卡片、評級、檢查清單、操作建議、基金資訊、資料 pills；PDF 跳過營收/EPS/PE 估值章節，顯示 NAV/折溢價/費用率/AUM 及 ETF 專有名詞）
- **SSE 斷線防護**（done 旗標防止重連觸發「任務不存在」錯誤）
- **資料新鮮度標示**（每項資料顯示最後更新日期）
- **資料快取**（營收/EPS 自動快取 1 天，Preview 快取 5 分鐘）
- **上櫃股票支援**（自動使用 .TWO 後綴）
- **貓裝飾邊框**（隨機貓圖側邊 + GIF 動畫邊條四邊環繞）
- **瀏覽器關閉自動結束程式**（sendBeacon + /shutdown）
- **Enter 鍵觸發分析**（輸入框按 Enter 等同按開始分析）
- **回報問題連結**
- **白色貓咪 SVG**（新手教學彈窗顯示）
- **投資免責聲明**

## 需求環境

- Python 3.8+
- Windows / macOS / Linux 均可

## 安裝步驟

### 1. 安裝 Python 套件

```bash
cd tw_stock_analysis
pip install -r requirements.txt
```

若遇到 `externally-managed-environment` 錯誤（Linux/WSL 常見），請改執行：

```bash
pip install --break-system-packages -r requirements.txt
```

### 2. 確認字型

PDF 的中文字型使用**微軟正黑體**（msjh.ttc），已內建於 `fonts/` 目錄。

若在 Linux 環境且沒有 Windows 字型，請安裝：

```bash
sudo apt install fonts-noto-cjk
```

## 執行方式

### Web 圖形介面（建議）

```bash
python webui.py
```

自動開啟瀏覽器前往 `http://127.0.0.1:5000`（可自訂埠號：`python webui.py 8080`）

支援功能：
- 輸入股票代號或名稱（支援部分比對，如 `233` → 2330 台積電）
- 即時顯示分析進度（視覺化進度條 + 分段百分比）
- Preview 預覽：
  - 綜合評級徽章（A/B/C/D，ETF 則為 ETF 評級）
  - 資料完整性 pills（彩色標示每項資料可用狀態）
  - 公司簡介（長描述、國家、員工人數）
  - 財務指標卡片（毛利率、營業利益率、淨利率、ROE、ROA、負債比、自由現金流、營收成長、Beta、預估本益比）
  - 近期新聞（前 5 則含情緒標籤與摘要）
  - ETF 模式獨立卡片集（NAV/折溢價/費用率/AUM/日均量/殖利率）
  - ETF 基金資訊（發行商＋類型）
  - 漲跌幅顯示（header + 主卡片）
  - 市值顯示（依規模自動換算）
  - 好壞範圍提示、名詞解釋彈窗
  - 檢查清單（✅/⚠️/❌，ETF 版本獨立）、操作建議（ETF 版本獨立）
  - 報酬預估、股市行事曆、新聞情緒、同業比較（含 6 欄對照表）
  - 風險警示分類（短期/中期/長期分組）
  - 股價/營收/EPS 圖表（ETF 隱藏營收/EPS 圖表）
- 合理價區間網格（便宜 / 合理 / 昂貴 / 目前，ETF 模式自動隱藏）
- 風險警示展開顯示（具體警示文字 + 紅/黃/綠燈）
- 股票比較（加入比較列表，多股並排）
- 歷史查詢（最近 10 筆，點擊重分析）
- 圖表切換（純股價 / 含均線 / 含成交量）
- CSV 匯出（單股 + 比較匯出）
- 英文介面切換（右上角 EN/中）
- 完成後一鍵下載 PDF
- 瀏覽器關閉自動結束程式
- Enter 鍵直接觸發分析

### 命令列模式

```bash
python main.py 2330
python main.py 台積電
python main.py TSMC
python main.py 2317 2454 2303
```

## 輸出

PDF 報告會產生在 `output/` 目錄，檔名格式：`{股票代號}_report_{日期時間}.pdf`

每份報告約 10–12 頁（個股）或 8–10 頁（ETF），內容包括：

| 頁面 | 個股模式 | ETF 模式 |
|------|---------|---------|
| 封面 | 股票名稱、代號、報告日期 | 同左 |
| 基本資訊 | 公司資料、股價、PE、PB、殖利率、ROE 等 | ETF 基本資訊（NAV、費用率、AUM、發行商、類型、日均量） |
| 股價走勢 | 3 個月 / 6 個月 / 1 年走勢圖 + 高點低點 | 同左 |
| 營收分析 | 每月營收圖表 + 近 6 期明細表 + 趨勢說明 | ╴（跳過） |
| EPS 分析 | 各季 EPS 圖表 + 明細表 + 年度總和 | ╴（跳過） |
| 估值分析 | 合理價區間、PEG、營收成長評估、健康度評分、風險提示 | ETF 評估指標（NAV/折溢價/費用率/AUM/殖利率） |
| 同業比較 | 股價 / 本益比 / 殖利率 | 股價 / NAV / 折溢價 / 費用率 |
| 股利分析 | 歷年股利表格、殖利率、連續配息年數 | 同左（ETF 配息紀錄） |
| 新聞摘要 | 近期相關公開新聞（標題、來源、日期） | 同左 |
| 專有名詞解釋 | PE/PEG/PB/ROE/殖利率/安全邊際/健康度 | NAV/折溢價/費用率/AUM/殖利率/追蹤誤差/PE |
| 免責聲明 | 投資風險提醒 | 同左 |

## 注意事項

1. **資料僅供參考**，所有投資決策應自行判斷。
2. 部分網站可能有反爬機制，若某來源失敗不會影響整體程式。
3. 歷史資料長度依各資料來源而定，非所有股票皆有完整數據。
4. 估值分析使用歷史 PE 區間法，不保證未來股價表現。
