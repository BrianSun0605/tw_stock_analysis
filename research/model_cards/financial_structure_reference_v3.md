# 台灣官方季度財務結構參考 v3

## 用途與輸出契約

此規則使用最新一季 TWSE／TPEx 普通產業公司官方財報，輸出可追溯的財務結構「參考分級」。它不是破產機率、信用評等、投資建議或股價預測；正式財務安全評級仍保留空白，直到完成台灣歷史驗證。

輸出只適用於一般公司與 TDR。金融、保險、證券、ETF、ETN、REIT 與會計結構明顯不同的商品不適用，系統必須停止分級並說明原因。

## 可取得資料與公式

TWSE／TPEx `ci` 端點能提供下列同季官方欄位：資產總額、流動資產、流動負債、保留盈餘、負債總額、營業利益與營業收入。因端點未提供原始 EBIT，v3 不宣稱為原始 Altman Z；它使用營業利益作為經營獲利能力代理，並將年初至今的損益欄位按 `4 / 已公告季度` 年化。

```text
Z-ref = 1.2 × (營運資金 / 總資產)
      + 1.4 × (保留盈餘 / 總資產)
      + 3.3 × ((4 / 季度) × 年初至今營業利益 / 總資產)
      + 0.6 × (市值 / 總負債)
      + 1.0 × ((4 / 季度) × 年初至今營收 / 總資產)
```

市值是最新可得市場資料；其餘欄位必須來自同一份官方季度財報。任何必要欄位缺失時，不以零、其他期間或其他比率補值。

## 參考分級

| 參考分級 | Z-ref 區間 | 顯示意義 |
|---|---:|---|
| A | 大於 2.99 | 財務結構相對穩定的參考訊號 |
| C | 1.81 至 2.99 | 需持續追蹤的參考訊號 |
| E | 小於 1.81 | 財務結構壓力較高的參考訊號 |

這些區間是透明的規則型參考，不是以台灣公司違約、下市或破產事件校準的機率。年化處理亦可能受季節性與一次性項目影響，因此不能單獨作為買賣依據。

## 覆蓋稽核與限制

2026-07-20 的官方端點稽核涵蓋 1,927 家上市／上櫃普通公司：資產負債表核心欄位與營業利益可取得，但原始 EBIT 為 0 家可取得。因此 v3 選擇明示代理欄位與免責聲明，而非無聲替代後冒充原始 Altman Z。

在取得公告日可回溯的年度／季度財報、明確定義的財務困境結果，以及按時間排序的樣本外驗證前，禁止把 A／C／E 改稱為正式台灣信用或破產評級。
