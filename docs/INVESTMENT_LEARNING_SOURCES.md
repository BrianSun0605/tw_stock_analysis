# 投資小教室：內容與來源查證

更新日期：2026-07-20

## 範圍與設計原則

投資小教室是離線、雙語的投資教育功能，共有 7 個主題軌、44 個概念與 220 題題目。內容涵蓋投資基礎、K 線與價格圖、財報與估值、ETF 與資產配置、產業研究、新聞媒體識讀，以及策略、行為與風險管理。

- 題目是以來源的可驗證定義、風險揭露與查證方法重新編寫，不直接複製來源文字。
- 題目避免提供買賣指令、預測價格、保證獲利或將技術指標當成單一交易訊號。
- 課程資料隨 App 發行，不會在使用者學習時向外部網站請求資料，也不會傳送學習進度。
- 每題在介面中附有來源連結；連結僅供使用者自行核對。
- 原本 12 個快速教學概念保留在新版對應主題中。本次課程紀錄版本更新會在首次載入時清除舊版完成與錯題紀錄；之後使用者可在課程首頁自行清除答題紀錄，且星號重點題目會保留。

## 課程架構

| 主題軌 | 核心範圍 | 主要查證原則 |
| --- | --- | --- |
| 投資基礎 | 目標、期限、風險、複利、分散與配置 | 個人期限與風險承受度先於商品選擇 |
| 圖表與價格行為 | 折線圖、K 線 OHLC、影線、成交量、趨勢、波動 | 圖表描述已發生資料，不單獨預測未來價格 |
| 財報、指標與估值 | 營收、EPS、P/E、利潤率、三大報表、殖利率、估值 | 指標需連同期間、假設與限制判讀 |
| ETF 與資產配置 | NAV、折溢價、費用率、追蹤誤差、流動性與集中度 | 基金市價、淨值、持有內容與成本必須一起看 |
| 產業研究 | 半導體、金融、循環、科技軟體、生技醫療 | 以產業專屬的觀察清單核對，而非單一數字 |
| 新聞與資訊識讀 | 原始揭露、標題脈絡、社群訊號、利益衝突、時間點、詐騙警訊 | 先找原始文件、日期與利益關係，再形成判斷 |
| 策略、行為與風險 | 計畫、部位、成本、交易紀律、確認偏誤、過度交易 | 用預先規則與反證檢查降低情緒化決策 |

## 納入與維護規則

1. 只使用監理機構、交易所或其投資人教育資料中可直接核對的定義與風險提醒作為主要依據。
2. 公式與情境題會由第二個概念性來源或原始文件的上下文交叉檢查；若資料不足，題目改為教導「資訊不足」而不猜測。
3. 任何帶有績效、價格方向或技術型態的內容，一律加入期間、假設、限制或「不是交易訊號」的界線。
4. 新增題目時，必須提供中英文內容、題目 ID、主題、難度、至少一個來源 ID 與常見誤解；缺一項不納入正式題庫。
5. 發布前執行前端測試，確認題庫數量、雙語欄位、來源連結、舊進度遷移與答案位置輪替均正常。

## 免責與限制

本功能只作教育與研究用途，不構成投資、稅務或法律建議。來源雖經查證，但金融商品、法規和個別公司資訊仍可能變動；使用者應查核最新官方揭露，並依自身目標、期限與風險承受度做決策。
