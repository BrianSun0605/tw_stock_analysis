# 隱私與網路連線說明

本專案有兩種明確不同的執行方式，資料路徑也不同。

## Windows 本機 App

- App 只在本機 `127.0.0.1` 提供介面，不接受區域網路或 Internet 連入。
- App 不要求帳號，不收集姓名、Email、交易紀錄或投資組合。
- 查詢的股票代號會由使用者電腦送到 TWSE、TPEx、公開資訊觀測站、Yahoo Finance 與新聞 RSS 等資料來源；這些來源可能依自己的政策記錄 IP、User-Agent 與請求。
- 分析 cache、SQLite、log 與 PDF 都留在本機。release build 使用每位 Windows 使用者的 LocalAppData；dev mode 使用專案目錄。
- PDF 預設逐檔保留三天，output 超過 250 MiB 時先刪最舊檔；cache 超過 200 MiB 時清到 160 MiB。
- 本機模式不會把分析結果上傳到本專案作者的伺服器。

使用者可從 App 的 output 目錄刪除 PDF；解除安裝程式應提供保留或移除使用者資料的選項。

## Render 公開展示網站

- 只有明確以 `TWSTOCK_APP_MODE=web` 部署時才是公開網站。使用者輸入的證券代號／名稱及分析請求會先送到 Render 上執行本專案的服務，再由該服務向上述資料來源抓取資料；外部資料來源通常看到的是主機的 IP，而不是使用者直接連線。
- 網站沒有帳號、登入、使用者資料庫、廣告追蹤或本專案主動加入的分析遙測。投資小教室的答題紀錄和星號題目仍只保存在使用者自己的瀏覽器 `localStorage`。
- 公開服務把工作資料、快取、操作 log 與 PDF 放在 Render 容器的暫存目錄。免費服務重新啟動、重新部署、休眠後喚醒或容量清理時，這些資料可能消失；PDF 應在產生後立即下載。
- 本專案不把分析結果寫入自行管理的永久資料庫；但 Render 與外部資料來源可能依各自政策保存連線 IP、存取 log、User-Agent 或請求中繼資料。公開部署前，管理者應審閱 Render 與各上游來源的現行隱私條款。
- 公開展示版會限制每個來源的搜尋與分析啟動頻率，並只執行一個重工作；這是資源保護，不是帳號驗證或完整的隱私／資料隔離機制。

完整 GitHub + Render 部署、額度與維運限制見 [DEPLOYMENT_RENDER.md](DEPLOYMENT_RENDER.md)。
